# CircleImageView
在 https://github.com/hdodenhof/CircleImageView 的基础上添加圆角矩形

通过type属性定义类型，默认为圆形图片（circle,roundRect两种类型）

圆角矩形通过round_rect_corner属性设置圆角大小，圆角矩形暂不支持border_overlay属性（边框是否覆盖在图片上）
